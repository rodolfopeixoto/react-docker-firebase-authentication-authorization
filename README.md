Set up

---------------------

Caso queira rodar localmente, basta seguir os passos abaixo.

Clone o repositório e de um:


 Basta rodar o comando abaixo:


 ```
  $ docker-compose build
 ```

 ```
  $ docker-compose run --rm web npm install
 ```

 ```
  $ sudo chown -R $USER:$USER .
 ```


 ```
  $ docker-compose up
 ```
